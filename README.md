# MiniGameRPG
### 미니게임과 2D RPG 게임을 결합한 모바일 게임
용사가 미니게임을 플레이하면서 적들을 물리치는 2D RPG 게임

### 🚀개발 동기
* 똑같은 양산형 게임으로 떨어진 신선함
  * 직관적이고 플레이타임이 짧은 미니게임을 활용하여 다채로운 RPG 게임 제작
* 데이터 관리 역량 향상
<br></br>
